Cillionaire - Let's make a Crypto-Millionaire

Cillionaire is a donation game where each participant contributes the same stake until a pre-defined pot limit is reached. Then, a winner is chosen randomly from the participants. In the event that the game needs to be cancelled, all participants are refunded.
The smart contract handles the ETH transfers and enforces the rules.
